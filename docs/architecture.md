# Architecture Overview

## High-level flow
1. Employee receives assignments.
2. Assignment/overtime/stress data is stored by backend.
3. Risk scoring job computes overload index.
4. HR dashboard surfaces high-risk employees.
5. Admin configures risk rules and governance.

## Security and roles
- ADMIN: full platform management.
- HR: monitor workload and rebalance assignments.
- EMPLOYEE: view own tasks, upload feedback.
